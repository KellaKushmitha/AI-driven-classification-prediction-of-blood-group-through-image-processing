# AI-driven-classification-prediction-of-blood-group-through-image-processing
AI-Driven Blood Group Classification using Image Processing, Deep Learning Neural Networks (DNN), and Firefly Optimization (FFO). The system preprocesses blood sample images and predicts blood groups with over 98% accuracy, offering a fast, efficient, and scalable healthcare diagnostic solution.

## Overview
This project predicts blood groups using image processing and deep learning techniques. The system analyzes blood sample images and classifies them into different blood groups.

## Features
- Blood group prediction
- Image preprocessing
- Deep learning classification
- Django-based web interface
- Dataset training and testing
